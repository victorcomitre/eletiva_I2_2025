Sistema de Controle de Exposições de Arte: Gerenciamento de exposições de arte em galerias e museus (ARTControl).

## Requisitos Funcionais:
- RF1 - CRUD para cadastro de artistas.
- RF2 - CRUD para cadastro de obras de arte (título, técnica, data de criação)
- RF3 - CRUD para cadastro de exposições (data, galeria, tema).
- RF4 - Curadoria: registrar participação de obras em exposições, associando artistas a obras.

## Tecnologias
- Back-end: PHP 8+.
- Banco de Dados: MySQL.
- Front-end: HTML5, CSS3, Bootstrap 5.
- Ícones: Bootstrap Icons.

## Como rodar o projeto
1. Clone o repositório.
2. Importe o arquivo `banco.sql` no seu gerenciador de banco de dados (phpMyAdmin/Workbench).
3. Configure o arquivo `conexao.php` com as credenciais locais.
4. Inicie o servidor PHP (php -S localhost:8080)
